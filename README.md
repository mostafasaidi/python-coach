# Python Coach Bot

یک ربات تلگرام برای آموزش پایتون به زبان پارسی، طراحی شده برای بازار کار آلمان.

## ویژگی‌ها
- استفاده از Vercel Serverless Functions
- webhook-based (بدون polling)
- ذخیره پیشرفت کاربران در Vercel KV
- ۱۲ مرحله یادگیری سختگیرانه
- یادآوری روزانه
- فقط به زبان پارسی صحبت می‌کند

## مراحل یادگیری
1. پایه‌های پایتون
2. ساختار داده‌ها
3. کنترل جریان
4. توابع
5. برنامه‌نویسی شی‌گرا
6. ماژول‌ها و بسته‌ها
7. ورودی/خروجی فایل
8. استثناها
9. تست‌نویسی
10. توسعه وب
11. آماده‌سازی برای AWS
12. آماده‌سازی شغلی

## نحوه استقرار در Vercel

1. این ریپوزیتوری را در GitHub آپلود کنید.

2. در [Vercel](https://vercel.com) یک پروژه جدید ایجاد کنید و به ریپوزیتوری GitHub متصل کنید.

3. یک دیتابیس Vercel KV ایجاد کنید:
   - در داشبورد Vercel، به Storage > KV بروید.
   - یک دیتابیس جدید ایجاد کنید.
   - KV_URL را کپی کنید.

4. متغیرهای محیطی را تنظیم کنید:
   - TELEGRAM_BOT_TOKEN: توکن ربات تلگرام شما (از @BotFather دریافت کنید)
   - DEEPSEEK_API_KEY: کلید API DeepSeek
   - KV_URL: URL دیتابیس KV

5. پروژه را deploy کنید.

ربات به طور خودکار webhook را تنظیم می‌کند و یادآوری روزانه را ارسال می‌کند.
